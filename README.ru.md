# AI Studio No Sleep

Язык: [English](README.md) | **Русский**

Безопасное и ориентированное на конфиденциальность расширение, разработанное специально для **Google AI Studio** (`aistudio.google.com`). Оно обеспечивает стабильную работу длительных задач по генерации текста и кода в фоновом режиме, даже если окно браузера полностью свернуто, перекрыто другими окнами или когда вы переключаетесь на другие активные приложения.

## 🧠 Проблема и решение

### Проблема «засыпания» вкладок в фоновом режиме
Современные браузеры на базе Chromium (такие как Microsoft Edge и Google Chrome) используют агрессивные алгоритмы энергосбережения и оптимизации памяти. Когда вкладка уходит в фоновый режим или сворачивается:
1. **Ограничение CPU (Throttling):** Очереди таймеров JavaScript (`setInterval`/`setTimeout`) ограничиваются до 1 срабатывания в минуту или приостанавливаются полностью.
2. **Остановка rAF:** Браузер полностью останавливает цикл `requestAnimationFrame` (0 Гц), что нарушает рендеринг входящих потоков текста.
3. **Замораживание Angular CDK Virtual Scroll:** Так как Google AI Studio использует виртуальный рендеринг Angular, уход курсора из активной зоны блокирует обновление DOM. Генерация приостанавливается и возобновляется только при ручной активации вкладки (что вызывает задержку «холодного старта»).

### Наше архитектурное решение
"AI Studio No Sleep" обходит эти ограничения с помощью скоординированного многоуровневого подхода:
* **Приоритет воспроизведения инфразвука (хак 1 Гц):** Воспроизводит математически синтезированную **инфразвуковую волну частотой 1 Гц** на громкости `0.02`. Она абсолютно неслышима для людей и домашних животных, но заставляет Chromium регистрировать вкладку как активный медиаплеер. Вкладке присваивается статус **Media Priority**, что освобождает ее основной поток от ограничений процессора.
* **Событийная прокрутка Angular CDK:** Переопределяет механику прокрутки путем отправки нативных событий `scroll` и симуляции нажатия аппаратной клавиши `End` на активном вьюпорте. Это заставляет Angular обрабатывать изменения и рендерить поступающий контент.
* **Постоянные порты обмена сообщениями (Keep-Alive):** Создает непрерывные фоновые каналы связи для предотвращения перехода сервис-воркера расширения в спящий режим.

## 🌟 Основные возможности

| Возможность | Описание | Техническая реализация |
| :--- | :--- | :--- |
| **Инфразвуковой приоритет** | Предотвращает засыпание браузера при сворачивании окна. | Генератор синусоидальной волны 1 Гц |
| **Блокировка спящего режима** | Препятствует отключению монитора и переходу ОС в спящий режим во время тяжелых задач. | HTML5 Screen Wake Lock API |
| **Адаптивная прокрутка** | Автоматически удерживает прокрутку в самом низу в режиме реального времени. | Диспетчер событий Angular CDK DOM |
| **Защита от случайного закрытия** | Предупреждает перед закрытием вкладки только во время активной генерации. | Перехват события на стадии `beforeunload` |
| **Динамический индикатор** | Отображает активный статус (`ON`) на иконке и меняется на галочку (`✓`) при завершении. | Контроллер бейджей `chrome.action` |
| **Системные уведомления** | Отправляет нативное уведомление ОС с точным временем выполнения задачи. | `chrome.notifications` API |
| **Незаметная интеграция** | Полностью сохраняет оригинальный дизайн интерфейса Google AI Studio. | Изолированный скрипт контента (DOM-isolated) |

## 🚀 Инструкция по установке

### Microsoft Edge и Google Chrome (режим разработчика)

1. **Скачайте исходный код:**
   Убедитесь, что папка `persistent-engine-ext` находится на вашем локальном диске.

2. **Откройте страницу расширений:**
   * В **Microsoft Edge** перейдите по адресу `edge://extensions/`
   * В **Google Chrome** перейдите по адресу `chrome://extensions/`

3. **Включите режим разработчика:**
   Активируйте переключатель **Режим разработчика** (обычно находится в левом боковом меню или в правом верхнем углу).

4. **Загрузите расширение:**
   * Нажмите кнопку **Загрузить распакованное** (Load unpacked).
   * Выберите папку `persistent-engine-ext`.

5. **Закрепите на панели инструментов:**
   Нажмите на иконку пазла (Расширения) на панели инструментов и закрепите **AI Studio No Sleep**, чтобы видеть индикатор состояния.

## ⚙️ Дополнительная настройка на уровне ОС

На некоторых ноутбуках системные профили энергосбережения могут переопределять политики браузера. Для гарантированной фоновой работы рекомендуется добавить постоянное исключение в настройках браузера:

1. Откройте настройки Edge, перейдя по адресу `edge://settings/system`
2. Прокрутите вниз до раздела **Оптимизация производительности**.
3. Найдите список **Никогда не переводить эти сайты в спящий режим**.
4. Нажмите кнопку **Добавить** и введите: `aistudio.google.com`
5. Нажмите **Добавить** для сохранения.

## 🎨 Пользовательский интерфейс

Интерфейс всплывающего окна (popup) спроектирован так, чтобы быть чистым, современным и понятным:

* **Двуязычный переключатель:** Быстрое переключение между английским и русским языками с помощью стилизованных круглых SVG-иконок флагов.
* **Неоновый пульс-монитор:** Включает в себя технологичную светящуюся линию сканирования и вертикальный эквалайзер. При активной работе волна динамически колеблется. В режиме ожидания анимация замирает и становится серой.
* **Сворачиваемые дополнительные параметры:** Сложные настройки подсистем (симуляция мыши, фоновый аудиоприоритет и ускорение CPU) скрыты под раскрывающейся панелью, чтобы не перегружать основной интерфейс.

## 📁 Архитектура проекта

Папка расширения содержит следующие модули:

```bash
persistent-engine-ext/
├── manifest.json       # Конфигурация манифеста MV3, привязанная к aistudio.google.com
├── content.js          # Скрипт страницы: инфразвук, автопрокрутка и защита от закрытия
├── background.js       # Фоновый скрипт: блокировка сна системы, порты связи и бейджи
├── popup.html          # Панель управления (темная тема) с SVG-флагами
├── popup.js            # Логика UI, переключение языков и анимация состояния
└── icons/
    ├── icon16.png      # Иконка расширения для панели задач (16x16)
    ├── icon48.png      # Средняя иконка (48x48)
    └── icon128.png     # Иконка высокого разрешения (128x128)
```

### Скрипты автоматизации развертывания
В корневой директории подготовлены вспомогательные скрипты CommonJS:
* `setup.cjs` — устанавливает зависимости проекта и генерирует чистую структуру каталогов.
* `rollback.cjs` — возвращает рабочую область к чистому состоянию коммита git при необходимости.
* `dump-current.cjs` — создает консолидированный файл `dump.txt`, содержащий текущее состояние всех активных файлов для аудита.

## 🔒 Конфиденциальность и безопасность

* **Строгая изоляция:** Расширение работает исключительно на сайте `https://aistudio.google.com/*`. Оно не запрашивает доступ к другим веб-ресурсам, сохраняя ваши персональные данные в безопасности.
* **Отсутствие аналитики:** В проекте нет трекеров, внешних аналитических систем или отправки телеметрии. Все настройки сохраняются локально на вашем компьютере через `chrome.storage.local`.
* **Абсолютная тишина:** Инфразвуковая волна работает на частоте 1 Гц, что полностью исключает слышимый человеком или животными шум и не вызывает дискомфорта.
